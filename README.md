## Quick Install

```bash
curl -fsSL https://raw.githubusercontent.com/marcogll/macbook_setupS23/main/install_macos.sh | bash
```

Or clone and run locally:

```bash
git clone git@github.com:marcogll/macbook_setupS23.git && cd macbook_setupS23 && ./install_macos.sh
```
